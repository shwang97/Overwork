Replication Package: Labor laws as family policy 
Author: Sungsik Hwang and Jenna Nobles
Contact: shwang97@wisc.edu
Date: 2025-11-18

The replication package contains five separate files 
1. OW_Cleaning: prepares the data for the analysis.
2. OW_MainAnalysis: Creating Figures/Tables in the main analyses.
3. OW_Robustness: Creating Figures/Tables in the appendix. 
4. OW_Falsification: Falsification test based on permutation.
5. Imputation: Impute missing values in working hours using the Amelia package in R.
